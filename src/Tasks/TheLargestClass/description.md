# Задание: Самый большой класс (while)

Школьники организованными группами заходят в музей. Каждая группа соответствует своему классу и школе. Все школьники одного класса проходят в музей последовательно и при этом называют класс и школу (эти числа вводятся в консоли). Необходимо определить какой класс и какой школы оказался самым многочисленным и вывести его размер (количество человек).

На вход с клавиатуры в консоли вводятся целые числа для каждого школьника - номер класса, номер школы. Считать, что пары класс - школа однозначно определяют одну группу (то есть допустим двух 7 классов из одной школы быть не может). Признак окончания ввода - 0.

На выходе должно быть число - количество школьников в самом многочисленном классе.

Результат: 6

- Массивы или другие структуры для хранения данных использовать не нужно.
- Необходимо использовать цикл с предусловием или постусловием.
- Вводится как минимум один школьник.
